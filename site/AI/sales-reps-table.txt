# UI Guide for Personal Accounts

![Labeled UI Screenshot](assets/ui-labeled.png)

This guide explains the functionality of each numbered UI element in the Personal Accounts section of the My Wallet application.

1. **My Wallet Logo**: Displays the application’s branding. Clicking it typically navigates back to the homepage or dashboard.
2. **User Email**: Shows the logged-in user’s email address (e.g., `retselistsoe@smartelmoney.com`).
3. **Menu Icon (Hamburger)**: Opens or closes the sidebar navigation menu.
4. **Settings Icon (Gear)**: Accesses user or application settings.
5. **Resetlistsoe Label**: Likely a user-specific identifier or role label, possibly clickable for profile details.
6. **Dashboard Menu Item**: Navigates to the main dashboard of the application.
7. **Districts Menu Item**: Takes the user to a section for managing or viewing districts.
8. **Personal Accounts Menu Item**: Currently selected, displays the personal accounts management section.
9. **Agent Accounts Menu Item**: Navigates to a section for managing agent accounts.
10. **Messages Menu Item**: Opens the messaging or notifications section.
11. **MRTO Module Menu Item**: Accesses the MRTO (possibly a custom module) section.
12. **Sales Reps Module Menu Item**: Takes the user to a sales representatives management section.
13. **VAS Module Menu Item**: Navigates to the Value-Added Services (VAS) module.
14. **NPS Menu Item**: Opens the Net Promoter Score (NPS) section for feedback or surveys.
15. **Mokhatlo Menu Item**: Likely a custom module or section (possibly a local term), navigates to related features.
16. **Personal Accounts Header**: Indicates the current section, showing you’re in the Personal Accounts area.
17. **Search National ID Field**: Allows searching for accounts by National ID (e.g., input like `1,2 c`).
18. **Search Cell Number Field**: Enables searching for accounts by cell number (e.g., input like `8 d`).
19. **Apply Button**: Applies the search filters entered in the National ID or Cell Number fields.
20. **Search Input Field**: A general search bar to filter table results by keywords.
21. **Refresh Icon**: Refreshes the table data to show the latest information.
22. **Download Icon**: Downloads the table data as a file (e.g., CSV or Excel).
23. **Table Headers**: Columns defining the data fields: First Name, Surname, ID/Passport Num, Cell, District, Area, Date, and Gender.
